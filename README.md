🧑‍💻 About

This is my personal developer portfolio website built using modern technologies to showcase:
	•	💻 My Projects
	•	⚡ Technical Skills
	•	📄 Resume
	•	📬 Contact Information

The goal is to present my work with smooth animations, clean UI, and high performance.

⸻

✨ Features
	•	🎬 GSAP animations (ScrollTrigger, ScrollSmoother, SplitText)
	•	⚡ Smooth scrolling experience
	•	🎯 Interactive UI/UX
	•	📱 Fully responsive design
	•	📂 Project showcase section
	•	📄 Resume download
	•	🎨 Clean and minimal design

⸻

🛠️ Tech Stack
	•	⚛️ React + TypeScript
	•	⚡ Vite
	•	🎬 GSAP
	•	🎨 CSS3
	•	🌐 Vercel
